# Aseprite compilation assistant.
Template to compile Aseprite. If you don't know what this is, then I encourage you to try it. It's a way to get Aseprite for free.

_____________________________________
* To run simply type in your terminal:

cd ~/Desktop/compile-aseprite (or compile-aseprite-main, if working with a .zip file) then type ./main.sh

Then follow the prompts.
